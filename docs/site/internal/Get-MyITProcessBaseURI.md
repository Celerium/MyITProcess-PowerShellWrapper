# Get-MyITProcessBaseURI

## SYNOPSIS
Shows the MyITProcess base URI global variable.

## SYNTAX

```powershell
Get-MyITProcessBaseURI [<CommonParameters>]
```

## DESCRIPTION
The Get-MyITProcessBaseURI cmdlet shows the MyITProcess base URI global variable value.

### EXAMPLE 1
